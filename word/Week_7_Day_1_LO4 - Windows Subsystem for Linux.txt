Introduction
3
Windows Subsystem forLinux (WSL) isafeature ofWindows that allows
developers torunaLinux environment without needing aseparate virtual
machine ordualbooting .
There aretwoversions ofWSLnamely WSL1andWSL2.
WSL1wasfirstreleased in2016 andacted asacompatibility layer forrunning
Linux binary executables byimplementing Linux system calls ontheWindows
kernel .
WSL 1isavailable onWindows 10,Windows 11,Windows Server 2016,
Windows Server 2019 andWindows Server 2022.Introduction Cont.
4
WSL2introduced important changes such asarealLinux kernel .
WSL2differs from WSL1inthatWSL2runsinside amanaged virtual machine
that implements thefullLinux kernel .Asaresult, WSL2iscompatible with
more Linux binaries, asnotallsyscalls were implemented inWSL1.
Since June 2019,WSL2hasbeen available toWindows 10customers through
theWindows Insider program, including theHome edition .
WSLisnotavailable toallWindows 10users bydefault .Itcanbeinstalled
either byjoining theWindows Insider program orbymanual installation .Introduction Cont.
